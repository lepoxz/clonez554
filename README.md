# FB88 MMO

Frontend cua du an `FB88 MMO | He thong cung cap tai nguyen & tool mmo`.

## Cau truc hien tai

- `app/`: route va app router cua Next.js
- `components/`: cac component giao dien dung lai
- `services/`: logic lay va bien doi du lieu
- `public/`: favicon, manifest va tai nguyen public
- `backend/`: thu muc placeholder cho backend sau nay

## Chay frontend

```bash
npm install
npm run dev
```

## Ghi chu

- Frontend hien tai la Next.js 15
- Khong commit `node_modules/`, `.next/`, `.env.local`
- Khi bo sung backend, giu tach biet ro logic frontend va backend
