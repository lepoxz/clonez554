# Backend

Thu muc nay duoc de san cho phan backend sau nay.

De xuat cau truc khi bat dau:
- `src/` cho source code server
- `src/modules/` cho cac module nghiep vu
- `src/shared/` cho util, middleware, config dung chung
- `tests/` cho test backend

Hien tai repo nay dang chua frontend Next.js o root project.
